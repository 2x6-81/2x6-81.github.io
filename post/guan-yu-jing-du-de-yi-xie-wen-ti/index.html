关于精度的一些问题

2021-02-06

有时，浮点数的精度是做题时的一个重要的问题。举一个例子：

#include <stdio.h>
main() {
    printf("%.6lf\n", floor(sqrt(9999.9999) * sqrt(10000.0001)));
}

这份代码是一个简单的根号的乘法，我们可以手算得到 9999.9999×10000.0001=100002−0.00012<1000\sqrt{9999.9999 \times 10000.0001} = \sqrt{10000^2 - 0.0001^2} < 10009999.9999×10000.0001​=100002−0.00012​<1000，但是当你真正运行的时候，会发现它会输出 10000.000000 ，而不是 9999.000000 。这是为什么呢？
